

<div class="head"><h1>User Persona</h1></div>

````ad-abstract
Se trata de un personaje ficticio creado para representar a un tipo de usuario que podría utilizar un sitio, una marca o un producto de forma similar.

User Personas son útiles para tener en cuenta los objetivos, deseos y limitaciones de los compradores y usuarios de la marca con el fin de ayudar a orientar las decisiones sobre un servicio, producto o espacio de interacción como las características, las interacciones y el diseño visual de un sitio web.
````
