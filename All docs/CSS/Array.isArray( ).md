# Array.isArray( )
<hr> 

> [!abstract]
> El método `**Array.isArray()**` determina si el valor pasado es un `Array`
> ```js
> Array.isArray(obj)
> ```
> 
> > [!example]
> > 
> > ```js
> > Array.isArray([1, 2, 3]);  // true
> > Array.isArray({foo: 123}); // false
> > Array.isArray('foobar');   // false
> > Array.isArray(undefined);  // false
> > ```
> >
