<h1>formmethod</h1>

> [!abstract]
> Defines the HTTP method for sending form-data to the action URL. Note: This attribute overrides the method attribute of the element.

<hr>

> [!example]
> <input type="submit" formmethod="post" value="Submit using POST">	
> 
> > [!bug] Code
