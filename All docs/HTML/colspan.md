# colspan
> [!abstract]
> ````
> You can have a cell that spans over two or more columns. ^a45101
> 
> <hr>
> ad-example

	<tr>
		<th colspan="2">Persons</th>
	</tr>
	<tr>
		<td>Emily</td><td>Tobias</td>
	</tr>
