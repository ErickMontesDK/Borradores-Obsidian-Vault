# type list
> [!abstract]
> ````
> Attribute of the tag, defines the type of the list item marker.
> For ordered lists. ^6489e9
> 
> <hr>
> ad-example
<ol type="i">
	<li>Coffee</li>
	<li>Tea</li>
	<li>Milk</li>
</ol>

> [!bug] Code
> ~~~html
> <ol type="i">
> 	<li>Coffee</li>
> 	<li>Tea</li>
> 	<li>Milk</li>
> </ol>
> ~~~


````

<hr> 

> [!tip]
> _a_: lowercase letters
> 
> _A_: uppercase letters
> 
> _i_: lowercase Roman numerals
> 
> _I_: uppercase Roman numerals
> 
> _1_: numbers
