# Math.random()
<hr> 

> [!abstract]
> Devuelve un número de coma flotante pseudo-aleatorio, comprendido en el rango de 0 a menor que 1
> ```js
> Math.random();
> ```
> 
> > [!example]
> > 
> > ```js
> > console.log(Math.random());
> > //Number between 0 and 1
> > ```
> > 
> >
