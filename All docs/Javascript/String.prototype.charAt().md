# String.prototype.charAt()
<hr> 

> [!abstract]
> The `charAt()` method returns the character at a specified index (position) in a string
> ```js
> string.charAt(index)
> ```
> 
> > [!example]
> > 
> > ```js
> > let text = "HELLO WORLD";
> > let letter = text.charAt(0);
> > console.log(letter);
> > ```
> > 
> > > [!bug] Console
> > > <code>H</code>
> >
