# typeof
<hr> 

> [!abstract]
> You can use the `typeof` operator to find the data type of a JavaScript variable.
> > [!example]
> > ```js
> > let stringV = typeof "John"// Returns "string"  
> > let numberV = typeof 3.14// Returns "number"
> > console.log(stringV, numberV)
> ```
