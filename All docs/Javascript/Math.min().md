# Math.min()
<hr> 

> [!abstract]
> La función **`Math.min()`** devuelve el menor de cero o más números.
> ```js
> Math.min(var1,var2,var3,...varn)
> ```
> 
> > [!example]
> > 
> > ```js
> > console.log(Math.min(0, 150, 30, 20, 38));
> > ```
> > 
> > > [!bug] Console
> > > <code>0</code>
> > 
>
