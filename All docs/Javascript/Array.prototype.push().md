# Array.prototype.push()
<hr> 

> [!abstract]
> Append data to the end of an array is via the `push()` function.
> ```js
> arr.push(element1[, ...[, elementN]])
> ```
> 
> > [!example]
> > 
> > ```js
> >  const myArray = [ ["John", 23], ["cat", 2]]; 
> >  myArray.push(["dog", 3]);
> >  console.log(myArray);
> > ```
> > 
> > > [!bug] Console
> > > <code>[ ["John", 23], ["cat", 2], ["dog", 3]]</code>
> > 
>
