# Array.prototype.pop()
<hr> 

> [!abstract]
> El método `**pop()**` elimina el **último** elemento de un array y lo devuelve. Este método cambia la longitud del array.
> ```js
> arr.pop()
> ```
> 
> > [!example]
> > 
> > ```js
> > 	const threeArr = [1, 4, 6];
> > 	const oneDown = threeArr.pop(); //oneDown=6;
> > 	console.log(oneDown);
> > 	console.log(threeArr);
> > ```
> > 
> > > [!bug] Console
> > > <code>6
> > > [1, 4]</code>
> > 
>
