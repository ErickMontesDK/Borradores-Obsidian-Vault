# Delete Properties to a Object
<hr> 

> [!abstract]
> We can also delete properties from objects with <mark>delete</mark>
> ad-example
```js
const ourDog = {
  "name": "Camper",
  "legs": 4,
  "tails": 1,
  "friends": ["everything!"],
  "bark": "bow-wow"
};

delete ourDog.bark;
```
