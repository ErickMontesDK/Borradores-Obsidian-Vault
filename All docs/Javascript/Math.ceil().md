# Math.ceil()
<hr> 

> [!abstract]
> Devuelve el entero mayor o igual más próximo a un número dado. 
> ```js
> Math.ceil(numero)
> ```
> 
> > [!example]
> > 
> > ```js
> > console.log(Math.ceil(1.4));
> > ```
> > 
> > > [!bug] Console
> > > <code>2</code>
> > 
>
