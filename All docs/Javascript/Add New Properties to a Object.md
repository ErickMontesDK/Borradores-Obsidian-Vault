# Add New Properties to a Object
<hr> 

> [!abstract]
> 
> You can add new properties to existing JavaScript objects the same way you would modify them.
> 
> > [!example]
> > ```js
> > const ourDog = {
> >   "name": "Camper",
> >   "legs": 4,
> >   "tails": 1,
> >   "friends": ["everything!"]
> > };
> > 
> > ourDog.bark = "bow-wow"; //new propertie
> ```
