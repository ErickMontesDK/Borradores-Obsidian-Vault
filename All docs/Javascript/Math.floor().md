# Math.floor()
<hr> 

> [!abstract]
> Devuelve el máximo entero menor o igual a un número. La función devuelve el valor entero redondeado más bajo de la variable.
> ```js
> Math.floor(x) 
> ```
> 
> > [!example]
> > 
> > ```js
> > console.log(Math.floor(1.6)); //1
> > ```
> > 
> > > [!bug] Console
> > > <code>1</code>
> > 
>
