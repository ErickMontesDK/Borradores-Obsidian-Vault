> [!abstract]
> Objects are similar to `arrays`, except that instead of using indexes to access and modify their data, you access the data in objects through what are called <mark>properties</mark>
> 
> > [!example]
> > ```js
> > const cat = {
> >   "name": "Whiskers",
> >   "legs": 4,
> >   tails: 1,
> >   enemies: ["Water", "Dogs"],
> >   2:"Dos"
> > };
> ```
