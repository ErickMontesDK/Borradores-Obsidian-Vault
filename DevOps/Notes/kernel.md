<div class="head"><h1>kernel</h1></div>

````ad-abstract
**El Kernel o núcleo, es una parte fundamental del sistema operativo que se encarga de conceder el acceso al hardware de forma segura para todo el software que lo solicita**.

**El Kernel o núcleo de un sistema operativo <mark style="background: #ABF7F7A6;">sirve para administrar los recursos de hardware solicitados por los diferentes elementos de software y hacer de intermediario decidiendo a que y cuando se concede este acceso*</mark>* evitando así sobrecarga del sistema, recursos innecesarios y acceso a software malicioso al propio Kernel y llegar a poder controlar así todo el sistema. 
De este modo el Kernel <mark style="background: #ABF7F7A6;">sirve como elemento de seguridad</mark> teniendo que pasar por varias capas antes de poder tener acceso, <mark style="background: #ABF7F7A6;">además tiene que distribuir los recursos de manera eficiente y ordenada**</mark> para que el Hardware trabaje junto al Software de la mejor manera posible.
````
