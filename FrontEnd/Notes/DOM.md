<div class="head"><h1>DOM</h1></div>

````ad-abstract
DOM es la abreviatura de <mark style="background: #ABF7F7A6;">Document Object Model</mark>, que se refiere a la estructura jerárquica de objetos que representa una página web en un navegador. 

Es esencialmente una <mark style="background: #ABF7F7A6;">representación del contenido de una página web en forma de objetos que pueden ser manipulados mediante código JavaScript.</mark>

El DOM permite a los desarrolladores web acceder, manipular y actualizar el contenido y la estructura de una página web dinámicamente, lo que permite a los sitios web ofrecer una experiencia interactiva y personalizada para los usuarios. 

El navegador, al cargar una página, genera esta estructura en forma de árbol, a la que generalmente llamamos **DOM**. Cada vez que ocurre un cambio en los elementos de una página, el navegador actualiza esta estructura para reflejarlos.
````
