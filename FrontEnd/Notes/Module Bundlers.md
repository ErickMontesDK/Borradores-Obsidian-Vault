<div class="head"><h1>Module Bundlers</h1></div>

````ad-abstract
Herramienta utilizada en el desarrollo web para combinar múltiples archivos de código fuente en un solo archivo para su uso en el navegador. 

<mark style="background: #ABF7F7A6;">Los módulos son unidades de código reutilizable</mark> que se pueden importar y exportar en diferentes partes de una aplicación web.

<mark style="background: #ABF7F7A6;">Un module bundler toma todos los módulos que se utilizan en una aplicación web y los combina en un solo archivo optimizado y minificado</mark>, que se puede utilizar para el lanzamiento del sitio web. Esto ayuda a reducir el número de solicitudes que se hacen al servidor, lo que puede mejorar significativamente la velocidad de carga del sitio web.

Destaca [[Webpack]] como module bundler
````
